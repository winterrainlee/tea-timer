# 차 한 잔의 시간 · Time for Tea

끓는 물을 부은 뒤 수십 초의 짧은 시간 동안, 화면을 보든 안 보든 **"지금 따라낼 때"** 임을 시각·청각·화면으로 깨닫게 해 주는 모바일 차 우림 타이머.

- 설계 문서: [DESIGN.md](DESIGN.md), [design-v0.1.md](design-v0.1.md)
- 앱: `index.html` (바닐라 JS · PNG 도구 자산 + SVG 효과 · WebAudio)
- 도구 자산: `assets/vessels/*.png` (투명 PNG, 크로마키 원본은 `assets/vessels/source`)

## 실행

브라우저로 바로 열거나, 로컬 서버로 띄웁니다(서비스워커·오디오·Wake Lock은 `http(s)`에서 동작).

```bash
cd tea-timer
python3 -m http.server 8123
# 브라우저에서 http://localhost:8123 접속 (모바일은 같은 와이파이의 PC IP로)
```

홈 화면에 추가하면 전체화면 PWA로 실행됩니다.

## 사용

1. **우림 도구**를 고른다 — 서양 티팟 · 동양식 다호 · 개완 · 머그 · 표일배.
2. **차**를 좌우로 넘겨 고른다 — 녹차·백차·청향 우롱·생보이·농향 우롱·홍차·숙보이. (권장 온도 토스트, 권장 시간 자동 세팅)
3. **시간**을 슬라이더/±로 맞추고 **우리기 시작**.
4. 우러나며 막바지에 신호가 강해지고, **완료 5초 전 예고음 → 종소리·플래시**.
5. 도구별 완료 동작: 개완=뚜껑 들어 따르기, 표일배=버튼 눌러 내리기, 그 외=따라내기.
6. **다음 泡** 버튼으로 회차를 이어 우린다.

## M1 구현 범위

- 7종 차 · 5종 도구, 상태 머신(idle→pour→steep→done), 회차(泡) 수동 누적.
- 신호: 헤어라인 잔량 링(막바지 강조), 예고음(비례 축소)·초읽기 틱·완료 종소리(WebAudio 합성), 완료 플래시, 화면 유지(Wake Lock).
- PNG 도구 자산 기반 히어로 / 개완 찻물·찻잎 효과 / 표일배 버튼식 배수 / 불투명 도구 앰비언트 글로우 / 머그 수면색.
- `prefers-reduced-motion` 대응, 음소거 토글, 가로 스와이프로 차 전환.
