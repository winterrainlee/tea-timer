# 차 한 잔의 시간 · Time for Tea

[English](README.en.md)

차 우릴 때 타이머로 쓸 수 있는 모바일 웹앱입니다.
일반 스톱워치 앱으로도 시간을 잴 수는 있지만, 안 예뻐서 싫었습니다. 거기다 알람이나 예고 신호가 없어 잠깐 딴생각을 하면 놓치는 게 짜증났습니다. 그렇다고 15초, 20초짜리 우림을 위해 매번 휴대폰 타이머 앱을 열고 시간을 맞추는 것도 번거로워요.
그래서 만들었습니다.

## 아주 간략한 사용법

`차 한 잔의 시간`은 차 종류와 우림 도구를 고르고, 권장 시간에서 살짝 조정해 바로 시작하는 차 우림 타이머입니다.
우림이 진행되면 시간 타이머가 줄어들면서 다구 입구에서 증기가 나옵니다.
유리 다구를 사용할 경우 찻잎이 풀어지는 모습이나 찻물 색도 같이 볼 수 있습니다.
우림이 끝나기 몇 초 전부터 예고음과 초읽기 신호가 나오고, 완료되면 종소리와 화면 플래시로 "지금 따라낼 때"를 알려 줍니다.

## 주요 기능

- 7종 차 프리셋: 녹차, 백차, 청향 우롱, 농향 우롱, 홍차, 보이 생차, 보이 숙차
- 차를 고르면 다하 위에 마른 찻잎 미리보기 표시
- 5종 우림 도구: 서양 티팟, 동양식 다호, 개완, 머그, 표일배
- 차별 권장 시간과 권장 온도 안내
- 슬라이더와 `-`/`+` 버튼으로 빠른 시간 조정
- 완료 전 예고음, 초읽기 tick, 완료 종소리
- 화면 플래시와 Wake Lock으로 놓침 방지
- 표일배의 윗 챔버 우림 → 아래 잔 배수 흐름
- 설치 가능한 모바일 PWA 구조

## 사용 방법

1. 브라우저나 홈 화면 아이콘으로 앱을 엽니다.
2. 중앙의 우림 도구를 좌우 화살표나 스와이프로 고릅니다.
3. 차를 좌우 버튼으로 고릅니다.
4. 우림 시간을 조금 조정하고 `우리기 시작`을 누릅니다.
5. 예고음과 완료 신호가 나면 도구에 맞게 따라내거나 분리합니다.
6. 이어서 마실 때는 `다음 泡 우리기`로 다음 우림을 시작합니다.

## 실행

귀찮으시면 그냥 [이 사이트](https://winterrainlee.github.io/tea-timer/)로 들어오시면 됩니다.

혹시 나만의 사이트에서 실행시키고 싶다면 아래 과정을 참고하세요.

```bash
git clone https://github.com/winterrainlee/tea-timer.git
cd tea-timer
python3 -m http.server 8123
```

그다음 브라우저에서 `http://localhost:8123`을 엽니다.

GitHub에서 직접 수정해 보고 싶다면 먼저 이 저장소를 포크한 뒤, 포크한 저장소를 클론해서 실행하면 됩니다.
