## Generation Direction

- Refined semi-realistic app illustration.
- White ceramic for teapot/mug, zisha clay for eastern pot, clear glass for gaiwan/piaoyibei.
- No text, logo, cast shadow, floor, or reflection.
- Transparent final PNGs are intended to be layered over the dark timer UI with app-controlled glow, steam, liquor, leaves, ring, and shadow effects.
